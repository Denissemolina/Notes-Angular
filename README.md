# User stories

1- I should be able to create an account, log in and log out.
I as a user should be able to create an account and authenticate using Google login to access my notes.

2- I should be able to take notes
As a user I want to be able to take notes so I don't have to depend on my bad memory and I can always keep in mind the notes or important things that I used to write on paper.

3- I should be able to see the notes.
I as a user want to read my notes to remember what I wrote before.

4- I should be able to edit notes
As a user I want to be able to edit notes so that I can modify what I wrote before.

5- I should be able to delete notes.
I as a user want to delete a note so that I don't see it again.
